# Changelog

## 1.0.0 (2025-04-26)

### Features
- **Server Management** — Run, Debug, Stop Odoo with one click or keyboard shortcut
- **Module Management** — Update, Install, Uninstall modules with smart module picker (git-changed modules shown first)
- **Update Changed Modules** — Auto-detects modules with uncommitted git changes
- **Fast Update/Install** — Runs without debugger attached for speed, auto-restarts server in previous mode
- **Code Navigation** — Toggle between Model (.py) and View (.xml), Go to Model/XML ID/Function Definition
- **Word Under Cursor** — Navigation commands work on the word under cursor, no selection needed
- **Right-Click Context Menu** — Odoo navigation commands in editor context menu
- **Breakpoints Panel** — View, toggle, remove breakpoints from the sidebar
- **Debug Controls** — Continue, Step Over/Into/Out, Restart, Stop — appears when debugging
- **Log Filtering** — Filter terminal output by level (ALL/ERROR/WARNING/INFO/DEBUG) in real-time
- **JS Debugging** — Launch Chrome with debug port, auto-generate pathMapping for all custom modules
- **Scaffold Module** — Generate new module with manifest, models, views, security from template
- **Addons Path Manager** — Pick which directories to include as addons paths
- **Database Switcher** — Switch between PostgreSQL databases from status bar
- **Utilities** — Kill processes, Start PostgreSQL, Open Odoo in browser, Clear asset bundles, Remove unused imports
- **Keyboard Shortcuts** — Ctrl+Shift+R (Run), Ctrl+Shift+U (Update), Ctrl+Shift+T (Toggle), and more
- **Cross-Platform** — Works on Linux, macOS, and Windows
